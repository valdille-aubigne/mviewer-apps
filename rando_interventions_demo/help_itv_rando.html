Demande d'intervention sur les sentiers de randonnée

Cette carte vous permet de signaler des anomalies sur les sentiers de randonnée du Val d'Ille-Aubigné.

Le signalement s'effectue via le formulaire sur la gauche de l'écran. Merci de bien vouloir renseigner ;
  • la date d'observation de l'anomalie
  • une description du signalement
  • votre nom et votre mail (si vous souhaitez être recontacté)
  • une photo du signalement (si vous en avez une)
  • la localisation de l'anomalie (en plaçant un point sur la carte via le bouton dédié)
